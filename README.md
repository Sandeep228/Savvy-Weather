# Savvy-Weather



## [Live here](https://savvy-weather.vercel.app/)

## Key Features:
 - Get current weather condition
 - Also search and get other country weather condition

## Prerequisites

Node version 18.7.x



## Install packages

```shell
npm i
```

## Start the dev server

```shell
npm run dev
```

## Build the page

```shell
npm run build
```
